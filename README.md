# React Auth UI

A simple authentication UI built with React, Tailwind CSS, shadcn/ui, and Sonner.

## Features

- Login & Register modes
- Dynamic form switching
- API integration using Fetch
- Toast notifications with Sonner
- Minimal modern UI

## Tech Stack

- React
- Tailwind CSS
- shadcn/ui
- Sonner

## Run Locally

```bash
npm install
npm run dev
